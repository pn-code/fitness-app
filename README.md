# fitness-app

This fitness app was developed to access fundamental fitness tools required for achieve
success no matter your fitness standards.

Functionality:
    - Calorie Calculator
    - Diet and Exercise Planner
    - Food & Exercise Tracker


Technologies used:
    - HTML and CSS
    - JavaScript
    - React
